This software package introduces the Recursive World Language (RWL), an 
interpreted language for constructing large virtual scenarios aimed at 
simulating datasets for mobile robotics SLAM. 

- Build Dependencies: 
   * CMake
   * MRPT

 In Ubuntu: sudo apt-get install cmake libmrpt-dev 

- Included programs: 
   * rwl-compiler
